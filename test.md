Testing Markdown to PDF Conversion
==================================

The goal of this proof **of** concept is to take some fairly standard
markdown *and* generate a correctly formatted ***PDF*** document using
the wonderful Prawn library.

This includes:
--------------

  * Bulleted
  * List
  * Support

And of coure:
-------------

  1. Ordered
  2. List
  3. Support
